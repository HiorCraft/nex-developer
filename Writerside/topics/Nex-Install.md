# Installation

## Prerequisites

- A Minecraft server Paper 1.21.11 or higher
- Java Development Kit (JDK) 25 or
- Nex Plugin (latest version)

## Installation Steps
1. Download the latest version of the Nex Plugin from the official website or repository.
2. Stop your Minecraft server if it is currently running.
3. Place the downloaded Nex Plugin JAR file into the `plugins` directory of your Minecraft
4. server.
5. Start your Minecraft server to load the Nex Plugin.
6. Once the server is running, the Nex Plugin will initialize and create necessary configuration files in the `plugins/Nex` directory.
7. You can customize the plugin settings by editing the configuration files located in the `plugins/Nex` directory. Refer to the documentation for specific configuration options.
