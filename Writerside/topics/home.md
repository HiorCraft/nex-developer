# Nex System Documentation

Willkommen zum Nex System Dokumentation. Hier findest du alle Informationen, die du benötigst, um das Nex System zu verstehen und zu nutzen.

## Was ist das Nex System?
Nex System ist eine umfassende Plugins für Minecraft Server, die eine Vielzahl von Funktionen bietet, um die Verwaltung und das Spielerlebnis zu verbessern. Es bietet Tools für Projektmanagement, Aufgabenverwaltung, Kommunikation und vieles mehr.

## Hat Nex System eine API?

Ja, Nex System nutzt Mehre Apis, um eine Vielzahl von Funktionen bereitzustellen. Dazu gehören:

- **CommandAPI**: Für die Erstellung und Verwaltung von Befehlen.
- **LuckPerms**: Für die Verwaltung von Berechtigungen und Gruppen.
- **Surf-API**: Für die Logik und Verwaltung von Projekten und Aufgaben.

## Nex System hat auch Frameworks?

Ja, Nex System verwendet verschiedene Frameworks, um die Entwicklung und Wartung zu erleichtern. Dazu gehören:

- **Nex-Npc**: Für die Entwicklung von NPCs
